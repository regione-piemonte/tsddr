## Breadcrumbs

All'interno della specifica rotta definiamo un data che contiene i **breadcrumbs** e l'**header title**
Per esempio:

```
    breadcrumbs: [
        {
          label: 'CONTACTS.LIST.HEADER',
          href: '/contacts'
        },
        {
          label: 'CONTACTS.ITEM.HEADER',
          href: undefined
        }
      ],
    headerLabel: 'CONTACTS.ITEM.HEADER'
```
